Scripture Republishing Information

  • A list of redistributable (Public Domain and Creative Commons BY-NC-ND+) translations is at ftp://eBible.org/pub/Scriptures/translations.csv
  • The actual content of redistributable translations is at ftp://eBible.org/pub/Scriptures/ in files named translation-id_html.zip, translation-id_usfm.zip, and translation-id_usfx.zip.
  • The translation-id_html.zip file contains a set of files that, when unipped entirely together, preserving directory structure, is readable with a modern web browser (except maybe Opera or browsers other than Firefox running on Android, depending on the language).
  • Translation-id_usfm.zip files contain Unicode USFM files (not a Paratext backup, but importable into Paratext with manual metadata entry).
  • Translation-id_usfx.zip files contain usfx.xml Scripture contents, translation-id-VernacularParms.xml, and translation-id-VernacularAdditional.xml. The latter two files contain metadata (information about the translation) like library card information, copyright, license terms, etc.
  • The same information is kept on some non-redistributable translations, but not published in public view. The main difference is that a flag indicating that it is not redistributable is set.
  • The contents of the above .zip files are digitaly signed. See signature.txt and use sha1sum and Gnu Privacy Guard to check the signature.
  • Free/libre open source software Haiola is available at Haiola.org. It will eventually include most of the features of WordSend.
An old manually-maintained index of some Bible sites is at http://eBible.org/bible/.
Operational sites:
  • Pointers to many Bibles plus text of World English Bible, Cambodian Khmer, etc.: eBible.org
  • World English Bible and its variants: WorldEnglishBible.org, WeBible.org, and HNVBible.org
  • Papua New Guinea Scriptures: PNGScriptures.org and TokPlesBaibel.org
  • Pacific Area Bibles: PacificBibles.org
  • Vanuatu Bibles: VanuatuBibles.org / Baibol.org
  • Cambodian Khmer: BibleCambodia.org
Under construction, and not yet useful:
  • Bible translation index by language, focused on downloadable and online readable text, audio, and video (currently little more than an Ethnologue code lookup): Bible.cx
  • Home for Bibles in areas with Spanish and Portuguese as a national language: Sagradas-Escrituras.org
  • Home for Bibles in areas with French as a national language: LaSainteBible.org
