Haiola Downloads

You may download the current version of Haiola here:

File name Description
Haiola.dmg Mac disk image for setup on Mac OS X. Install mono, then drag Haiola.app to the Applications folder.
haiola.zip
 Manual setup files for Windows 7, Linux, or Mac OS X. Install mono and/or Microsoft .NET 3.5 SP1, then unzip to a folder and run from there. The folder can be on a flash drive or your hard disk drive. The manual part of the setup is creating startup scripts or icons yourself.
HaiolaSetup.msi
 Windows 7 installer. It should might also run on Windows Vista or XP, but I haven't tested that.
setup.exe Optional setup launcher for Windows 7.
haiola.htm Current documentation for Haiola.
