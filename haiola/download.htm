Haiola Downloads

You may download the current version of Haiola here:

File name Description
Haiola.dmg Mac disk image for setup on Mac OS X. Install mono, then drag Haiola.app to the Applications folder.
haiola.zip
 Manual setup files for Windows, Linux, or Mac OS X. Install a current Java Runtime Environment if you don't already have one. Install mono and/or Microsoft .NET 3.5 SP1, then unzip to a folder and run from there. The folder can be on a flash drive or your hard disk drive. The manual part of the setup is creating startup scripts or icons yourself.
haiolasetup.zip
 Windows installer. I build and test on Windows 8, but it should theoretically work on systems as old as Windows XP as long as the prerequisites are met. Use anything older than Windows 7 at your own risk. Note that a Java Runtime Environment (not included) is required.
haiola.htm Current documentation for Haiola.
