Haiola Scripture Publishing Software

Haiola is the continuation of both the Prophero (SEPP) and WordSend (Onyx) open source Scripture publishing software projects.

Haiola Scripture publishing software is in alpha test. So is this documentation. Please use it only at your own risk, and with frequent backups of your data.

Please remember that in publishing Scripture:

  • You must do your best to protect the integrity of the Holy Bible at all times, because it is God's Word.
  • You are responsible to comply with all applicable copyright and trademark requirements.
  • You are responsible to comply with applicable organizational guidelines concerning translation, checking, and publishing Scripture.

The current Haiola documentation assumes that you are very familiar with computers and file systems. Some knowledge of programming is helpful.

  • Haiola documentation
  • Haiola manual installation .zip file. Requires Mono. Note: if you need BibleBookInfo.xml or SFMInfo.xml for an older copy of WordSend, this is a good place to get it.
  • Zipped windows installation file. Requires Microsoft .NET Framework 3.5 to run.
  • Mac OS X disk image installation file. Drag Haiola.app to Applications directory to install. Requires Mono to run.
  • Change log and link to downloads
  • Haiola announcement email list sign-up or send a blank email to haiola-subscribe@eBible.org from the address you want to subscribe. (Nobody reads mail to that address. The Mailman program processes it and sends you a confirmation email.)
  • Haiola announcement Google group sign-up
  • Haiola source code repository (using the project's prior names, Prophero and SEPP)
  • Haiola source code repository mirror
